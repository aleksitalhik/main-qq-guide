# main-qq-guide

本仓库用于归档和发布多个独立 HTML 页面。页面内容涵盖日常参考、工具索引、信息整理等类目，不针对任何特定网站或域名。

## 项目简介

这是一个以 HTML 页面为主要载体的归档仓库。各页面彼此独立，可直接在浏览器中打开查看，适合用于存放轻量级的信息展示或工具型页面。仓库本身不提供后端服务或动态内容。

## 目录说明

- `pages/` —— 存放所有 HTML 页面文件，按功能或主题命名
- `assets/` —— 存放页面引用的静态资源（如 CSS、图片、字体等），可选
- `README.md` —— 本文件

页面文件均可直接通过 GitHub Pages 或本地文件系统访问，无需额外配置。

## 页面归档说明

- 每个 HTML 页面均为独立文件，不依赖其他页面或外部服务
- 页面内不包含跟踪代码、广告或外部请求（除非在说明中特别标注）
- 归档内容可能不定期更新或移除，不保证长期可用
- 欢迎提交 Issue 或 Pull Request 来报告问题或建议改进

## 维护说明

- 本仓库由个人维护，更新频率不定
- 不承诺对任何页面内容的持续更新或准确性
- 如需引用或分发，请保留原始出处说明
- 禁止将本仓库内容用于违反当地法律法规的用途

## 使用方式

1. 克隆或下载本仓库
2. 打开 `pages/` 目录下的任意 HTML 文件
3. 在浏览器中直接查看

也可通过 GitHub Pages 在线访问（如已开启）。
